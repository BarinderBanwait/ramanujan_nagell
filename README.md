## Ramanujan-Nagell in mathlib4

Formalizing the proof of the Ramanujan-Nagell theorem in Lean4, following the exposition in Stewart and Tall's book *Algebraic number theory and Fermat's Last Theorem*.
 The main file is [here](https://github.com/BarinderBanwait/ramanujan_nagell/blob/master/MyProject/RamanujanNagell.lean). Still a WIP ...
